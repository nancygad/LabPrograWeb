Sofia Pérez


Para concluir considero que todos estos conceptos ya seas ITS o IAS son conceptos que todo desarrollor web debe de saber puesto que son básicos para comenzar en el mundo de la tecnología, programación, etc. No importa si eres desarrollador de front-end, back-end o full-stack. Hay un lenguaje que cualquiera debe dominar al 100% y este es HTML el cual describe el contenido y define la estructura de una página web a través de marcas o etiquetas como lo indica su nombre. CSS es el otro lenguaje que debemos de dominar, pues las reglas que se definen en esta herramienta van a definir el comportamiento visual del sitio.

Html por si solo es una muy buena herramienta para elaboracion de paginas web, pero al hacer uso de los estilos, CSS, la creacion de paginas web se vuelve mas eficiente ya que se pueden agregar mucho mas funciones y etiquetas que haran que nuestra pagina web luzca mas elegante y formal. Sofia Pérez


Tras la aparición del lenguaje CSS, debido a que existen muchas formas de escribir código CSS, es necesario estandarizar el uso de todos los navegadores y desarrolladores de software. Además, los navegadores interpretan determinadas definiciones de estilo de diferentes formas, lo que dificulta el desarrollo web. De hecho, incluso con el mismo código, las personas involucradas en el desarrollo web a menudo se encuentran con situaciones en las que los navegadores muestran las páginas web de diferentes formas, simplemente por la forma en que el navegador interpreta el código. La organización responsable de la estandarización en esta área es el llamado W3C, que definió la primera versión de CSS1 en 1996. Posteriormente se desarrollaron las últimas revisiones 2, 2.1 y 3. El lenguaje CSS seguirá desarrollándose, pero hoy se puede decir que ha logrado un éxito en la simplificación del mantenimiento web. En el pasado, el contenido y los estilos de presentación se mezclaban. Con CSS, se pueden separar y el diseño y mantenimiento de las páginas web se vuelven más fáciles.
